👤 Author
Sourashis Mukherjee.


📖 Project Overview

This project is a Machine Learning-based Credit-wise Loan System that predicts whether a loan application should be approved or rejected based on applicant data. Multiple classification algorithms are implemented and compared to achieve the best performance.

🎯 Objectives 

1.Predict loan approval status using ML models
2.Compare different classification algorithms
3.Improve decision-making accuracy in loan systems
4.Automate the loan approval process

⚙️ Features 

1.Input applicant details (income, credit history, etc.)
2.Data preprocessing and cleaning
3.Multiple ML model predictions
4.Model comparison and evaluation
5.Final loan approval result

🧠 Machine Learning Models Used

1️⃣ Logistic Regression

1.Used for binary classification
2.Simple and interpretable model
3.Works well with linearly separable data
2️⃣ K-Nearest Neighbors (KNN)

1.Instance-based learning algorithm
2.Classifies based on nearest neighbors
3.Sensitive to feature scaling
3️⃣ Gaussian Naive Bayes

1.Based on probability and Bayes’ theorem
2.Assumes features follow a normal distribution
3.Fast and efficient for large datasets
🔍 Data Preprocessing

1.Handling missing values
2.Encoding categorical variables
3.Feature scaling (important for KNN)
📊 Model Workflow

1.Collect user input
2.Preprocess input data
3.Apply trained ML models:
Logistic Regression
KNN
Gaussian Naive Bayes
Compare predictions
Output final loan approval status
